## 문제

문제출처 : [Baekjoon][Baekjoon]

[Baekjoon]: https://www.acmicpc.net/problem/2438

![img](/image/bj_2438.PNG)

## 풀이1
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int num = sc.nextInt();
        sc.close();

        for(int i = 0; i < num; i++){
            for(int j = 0; j <= i; j++){
                System.out.print("*");
            }
            System.out.println(" ");
        }
    }
}
```
